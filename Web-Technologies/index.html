RESTful API Documentation

Create Request:

  • HTTP Method Type: POST
  • Request Format: http://localhost:3000/authors/
  • Request Body: A JSON object containing the details of the new author to be created. The object should include the following properties: "author", "alt", "tags", "image", and "description". Example:

    • { "author": "John Doe", "alt": "Author of the book 'Example'", "tags": "example, book, author", "image": "https://example.com/johndoe.jpg", "description": "John Doe is an accomplished author with several published works." }

  • Response Body: There is no response body
  • Response Code: 201 (CREATED) if the request is successful.
  • Response Code: 400 (Bad Request) if the request is unsuccessful.

GET Request

  • HTTP Method Type: GET
  • Request Format: http://localhost:3000/authors/
  • Request Body: Not required for GET requests
  • Response Body: A JSON object containing the requested author resource(s). Example:

    • { "author_id": 1, "author": "John Doe", "alt": "Author of the book 'Example'", "tags": "example, book, author", "image": "https://example.com/johndoe.jpg", "description": "John Doe is an accomplished author with several published works." }

  • Response Code: 200 (OK) if the request is successful.
  • Response Code: 500 (SERVER ERROR) if the request is unsuccessful.

Update Request:

  • HTTP Method Type: PUT
  • Request Format: http://localhost:3000/authors
  • Request Body: A JSON object containing the updated details of the author to be updated. The object should include the following properties: "author_id", "author", "alt", "tags", "image", and "description". Example:

    • { "author_id": 1, "author": "Jane Doe", "alt": "Author of the book 'New Example'", "tags": "new example, book, author", "image": "https://example.com/janedoe.jpg", "description": "Jane Doe is an accomplished author with several published works." }

  • Response Body: There is no response body
  • Response Code: 200 (OK) if the request is successful.
  • Response Code: 400 (Bad Request) if the request is unsuccessful.

Delete Request:

  • HTTP Method Type: DELETE
  • Request Format: http://localhost:3000/authors/{author_id}
  • Request: In the case of DELETE requests, no request body is necessary.
  • Response: The response body of a DELETE request is not necessary.
  • Upon successful deletion of the resource, the server should return a 204 (No Content) response code.
  • Response Code: 400 (Bad Request) if the request is unsuccessful.
